# Cidadania Digital e Inteligência Artificial
## O Impacto das Deepfakes e da Desinformação na Sociedade

### 🎯 Objetivo do Projeto
Este site foi desenvolvido como parte da Recuperação Prática do componente de Educação Digital e IA (Trimestre 1 - 2026). O principal objetivo da página é conscientizar e educar a comunidade escolar sobre os riscos associados às mídias manipuladas por algoritmos de Inteligência Artificial (Deepfakes) e fornecer estratégias práticas para identificar notícias falsas estruturadas de forma automatizada na internet.

### ⚙️ Tecnologias Utilizadas
- HTML5 (Estruturação semântica)
- CSS3 (Estilização avançada com Flexbox e responsividade para dispositivos móveis via Media Queries)
- JavaScript Vanilla (Lógica do validador do Quiz e alternador de Modo Escuro)

### 🤖 Uso de Inteligência Artificial (Créditos e Prompts)
Em conformidade com as regras éticas estabelecidas no projeto, a IA foi utilizada estritamente como assistente de coprodução de código. Seguem os comandos exatos utilizados:

- **Prompt para estruturação do código (Modelo Utilizado: AI Collaborator):**
  > "Atue como um desenvolvedor web sênior e gere uma página estruturada em HTML5 semântico, CSS3 responsivo com Flexbox e JavaScript Vanilla para um projeto escolar sobre Cidadania Digital, Deepfakes e Desinformação. O código deve incluir um botão funcional de modo escuro e um formulário/quiz interativo que valida as respostas do usuário diretamente na tela sem recarregar a página."
